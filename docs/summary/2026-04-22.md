위키 임베드 iframe 인증 연동, LFON 그룹웨어 계정 관리 자동화, 그리고 SSO JWT 발급 누락 핫픽스 등 3개 주요 기능을 완성 및 배포했습니다. 특히 전날 배포된 JWT 인증으로 인한 전면 401 회귀를 새벽 긴급 수정하여 운영 정상화했으며, 다우오피스 사용자 인증 불일치 이슈도 발견·해결했습니다.

## 주요 변경사항

### [추가] [Wiki/Embed] Outline 위키 iframe 임베드 인증 연동
- **모듈**: `frontend/app/embed/wiki/`, Outline Node.js 서버
- **배경**: 새벽 배포된 JWT 인증 의무화로 위키 iframe 채팅이 401 발생. Cross-origin iframe이라 쿠키 전달 불가
- **해결**: 그룹웨어 위젯과 동일한 **AES-128 ECB 암호화 토큰** 방식 재사용
  - Outline 측: `email|timestamp` 암호화 → iframe src `?token=` 파라미터
  - Lucid 측: `EmbedChat(widgetAuthToken)` → `X-Widget-Auth` 헤더 전송
  - 백엔드: `widget_auth.py`가 복호화 + email → login_id → 사번 조회
- **부수 수정**: `/embed` 경로의 Next.js trailing slash redirect loop 해결 (sub-path로 이동)
- **인텐트 제한**: `outline_embed` 모드는 OUTLINE + DIRECT만 허용 (위키 검색 범위만 접근)

### [추가] [IT/LFON] 그룹웨어 계정 관리 자동화 (OTP/비밀번호/메일용량 초기화)
- **모듈**: `backend/app/mcp_servers/lfon_mgmt/server.py`, `ITSupportWorker`
- **기능**: 사용자가 자연어로 "내 OTP 초기화해줘" / "비밀번호 리셋" / "메일 용량 늘려줘" 요청 시 2-step confirm/execute 패턴으로 안전 처리
- **구현**:
  - 신규 MCP 서버 (FastMCP stdio): 6개 도구 (confirm_* x3 + execute_* x3)
  - 사번 → `v_user_info_mapping.user_id` 변환 후 `/secure/lfon/management/*` PUT 호출
  - 토큰 1회성 + 60초 유효 + empno/action 이중 검증
  - `ITSupportWorker.SECURED_TOOLS`에 6개 추가하여 `prepare_tools()`에서 employee_number 강제 주입 (본인 계정만 조작 가능)
- **감사**: LFON 측이 원본 API 이력을 자체 DB에 기록하므로 챗봇 측 별도 감사 테이블 미도입

### [수정] [Security/Widget] 다우오피스 `${userId}` → 사번 매핑 불일치 해결

  - 사번/email/login_id 입력 모두 지원 (사번 패턴 `^[A-Z]\d{7}$` 매칭 시 pass-through)
  - 프로세스 수명 캐싱으로 DB 왕복 최소화
