# 2026-04-13 그룹웨어 플로팅 위젯 연동

## 개요
다우오피스 그룹웨어(LFON) 화면 내에 Lucid AI 챗봇을 플로팅 위젯 형태로 연동. 다우오피스 측 커스터마이징 없이, JSP 커스텀 삽입 포인트와 nginx 정적 파일 서빙을 활용하여 자체 구현.

## 변경 파일 요약
| 파일 | 변경 유형 | 설명 |
|------|-----------|------|
| `frontend/widget/lucid-chat-widget.js` | 수정 | SSE 이벤트 매핑을 백엔드 형식에 맞게 변경, SPA 환경 MutationObserver 추가 |
| `C:/Services/nginx/conf/conf.d/lfchatbot.conf` | 수정 | `/widget/` 정적 파일 서빙 location 추가 (HTTPS + HTTP) |
| `C:/Services/LFChatbot_prod/widget/` | 신규 | blue/green 전환에 독립적인 위젯 파일 고정 경로 |
| LFON 서버: `custom_index_header.jsp` | 수정 | 위젯 스크립트 삽입 (다우오피스 공식 커스텀 포인트) |

## 상세 내용

### 배경
- 다우오피스 고객케어에 공통 영역 JS 삽입 가능 여부 문의 → **"불가"** 회신
- 구축형(온프레미스) 서버 접근 권한을 활용하여 자체 해결

### 서버 구조 분석
```
사용자 → 다우오피스 내장 nginx(80/443) → Tomcat(8080)
         /data/TerraceTims/3rd/nginx/     다우오피스 WAS
```
- 내장 nginx: `sub_filter` 모듈 미포함 (최소 빌드) → nginx 레벨 JS 주입 불가
- JSP 분석 결과 `custom_index_header.jsp` 커스텀 삽입 포인트 발견

### LFON 서버 변경 (다우오피스)
`/opt/TerraceTims/web/webmail/webapps/ROOT/WEB-INF/views/webapp/custom_index_header.jsp`:
```jsp
<%@ page contentType="text/html;charset=UTF-8" language="java" %>
<!-- index.jsp에 대한 header 부분에 대한 custom 처리사항을 반영합니다. -->
<script src="https://lucid.landf.co.kr/widget/lucid-chat-widget.js"></script>
<script>
LucidChat.init({
    apiUrl: 'https://lucid.landf.co.kr',
    userId: '${userId}',
    position: 'bottom-right'
});
</script>
```
- `${userId}`: 다우오피스 JSP 변수 (로그인ID)
- SPA(Backbone.js) 구조로 최초 1회 로드 후 모든 화면에서 위젯 유지
- JSP는 Tomcat이 자동 리컴파일 → 서버 재시작 불필요

### Lucid 서버 변경

**nginx 설정** (`lfchatbot.conf`):
```nginx
location /widget/ {
    alias C:/Services/LFChatbot_prod/widget/;
    expires 1h;
    add_header Access-Control-Allow-Origin *;
    add_header Cache-Control "public, max-age=3600";
}
```
- HTTPS(443) + HTTP(80) 양쪽에 추가
- CORS: FastAPI `CORSMiddleware`가 `allow_origins=["*"]`로 처리 (nginx 중복 불필요)

**위젯 JS 수정** (`lucid-chat-widget.js`):
1. SSE 이벤트 매핑을 백엔드 `a2a_streaming.py` 형식에 맞게 변경:
   - 토큰 타입: `token` → `content`, 내용 필드: `content` → `chunk`
   - 도구 상태: `tool_call` → `tool_status`
   - 완료 이벤트: `{"type": "done"}` → `{"complete": true}` 감지 추가
2. API 경로: `/api/chat/stream` → `/api/v1/chat/message/stream` (백엔드 직접 호출)
3. MutationObserver 추가: SPA body 교체 시 위젯 자동 재삽입

### On/Off 운영
```bash
# OFF (위젯 숨기기) - Lucid 서버에서
mv C:/Services/LFChatbot_prod/widget/lucid-chat-widget.js C:/Services/LFChatbot_prod/widget/lucid-chat-widget.js.off

# ON (위젯 보이기)
mv C:/Services/LFChatbot_prod/widget/lucid-chat-widget.js.off C:/Services/LFChatbot_prod/widget/lucid-chat-widget.js
```
- LFON 서버 접속 불필요, 그룹웨어 재시작 불필요

## 결정 사항 및 주의점
- **다우오피스 업데이트 시 `custom_index_header.jsp`가 초기화될 수 있음** → 스크립트 3줄 재삽입으로 복구 (30초)
- **인증**: `${userId}`는 로그인ID, 백엔드에서 TIMS DB `v_user_info_mapping`으로 사번 변환 가능
- **현재 상태**: 위젯 OFF (승인 후 ON 전환 예정)
- **추가 비용**: 0원 (다우오피스 커스터마이징 요청 없음)
- **CORS**: FastAPI `allow_origins=["*"]`가 처리, nginx에서 중복 추가하면 `multiple values` 에러 발생 → nginx CORS 제거
