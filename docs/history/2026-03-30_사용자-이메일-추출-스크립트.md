# 2026-03-30 사용자 이메일 추출 스크립트

## 개요
2026-02-09 오픈 이후 챗봇을 사용한 전체 사용자의 이메일 주소를 추출하는 스크립트 작성. MySQL(chat_sessions)과 PostgreSQL(v_user_info_mapping)을 교차 조회하여 사번→그룹웨어ID→이메일 매핑.

## 실행 방법
```bash
cd backend
python -X utf8 c:\tmp\extract_user_emails.py
```

## 스크립트 위치
- `c:\tmp\extract_user_emails.py`

## 출력
- `c:\tmp\chatbot_user_emails.txt` — 콤마 구분 이메일 목록 (복붙용)

## 동작 흐름
1. MySQL `chat_sessions`에서 `created_at >= 2026-02-09` 고유 `user_id`(사번) 추출
2. PostgreSQL `v_user_info_mapping`에서 사번 → `login_id`(그룹웨어 ID) 매핑
3. `login_id` + `@landf.co.kr` 조합하여 이메일 생성
4. 정렬 후 콤마 구분 텍스트로 출력 및 파일 저장

## 설정값
| 항목 | 값 | 비고 |
|------|-----|------|
| 도메인 | `@landf.co.kr` | 고정 |
| 기준일 | `2026-02-09` | 오픈일, 필요시 변경 |
| MySQL | `backend/.env`의 DB_* 변수 | dotenv 로드 |
| PostgreSQL | `TIMS_DATABASE_URL` 또는 기본값 | ai_reader@192.168.100.5:5432/tims |
