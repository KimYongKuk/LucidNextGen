# 2026-04-14 그룹웨어 서비스 메뉴 위젯

## 개요
그룹웨어에 챗봇 플로팅 버튼과 별도로, SSO 연동 서비스(e-HR, EHS, wa, Lucid, Pilot, Hub 등)에 바로 접근할 수 있는 플로팅 서비스 메뉴 위젯을 추가했다. 사용자의 소속 조직(엘앤에프/엘앤에프플러스)에 따라 노출 메뉴가 자동으로 필터링된다.

## 변경 파일 요약
| 파일 | 변경 유형 | 설명 |
|------|-----------|------|
| backend/migrations/add_service_menu.sql | 추가 | service_menu 테이블 DDL + 초기 데이터 7건 |
| backend/app/api/routes/service_menu.py | 추가 | GET /api/v1/service-menu API (사번→조직→메뉴 필터) |
| backend/app/main.py | 수정 | service_menu 라우터 등록 |
| frontend/widget/lucid-service-menu.js | 추가 | 플로팅 서비스 메뉴 위젯 (vanilla JS) |

## 상세 내용

### 아키텍처
```
그룹웨어 페이지
  ↓ lucid-service-menu.js 로드
  ↓ LucidServiceMenu.init({ apiUrl })
  ↓
플로팅 버튼 클릭
  ↓ GO.session().employeeNumber 추출
  ↓ GET /api/v1/service-menu?empno=PA2601004
  ↓
백엔드
  ├─ PostgreSQL: go_users → go_companies (사번→소속 회사 조회)
  ├─ 회사명 프로세스 수명 캐싱 (_company_cache)
  └─ MySQL: service_menu 테이블 → 조직별 필터링
  ↓
메뉴 리스트 팝업 표시
  ↓ 사용자 클릭
  ↓ window.open(target_url, '_blank') — SSO JSP 경유
```

### DB 테이블 (MySQL - lucid)
```sql
service_menu (
  id, name, icon, target_url, orgs JSON, sort_order, enabled, created_at, updated_at
)
```

### 위젯 UI
- 보라색 그라데이션 버튼 (#6366f1 → #4338ca), 챗봇 버튼(파란색) 위에 배치
- 클릭 시 서비스 메뉴 팝업, 각 항목 클릭 시 새 탭으로 SSO URL 이동
- 외부 클릭 시 자동 닫힘
- 챗봇 위젯과 동일한 패턴: IIFE, `LucidServiceMenu.init()` API, MutationObserver SPA 대응

### 사용법 (그룹웨어 JSP)
```html
<script src="https://lucid.landf.co.kr/widget/lucid-service-menu.js"></script>
<script>
  LucidServiceMenu.init({ apiUrl: 'https://lucid.landf.co.kr' });
</script>
```

## 결정 사항 및 주의점
- 사번→회사 매핑은 프로세스 수명 캐싱 (직원의 소속 회사는 거의 변경되지 않음)
- 메뉴 추가/삭제는 MySQL service_menu 테이블에서 직접 관리 (코드 배포 불필요)
- nginx `/widget/` 경로가 이미 `C:/Services/LFChatbot_prod/widget/`를 서빙하므로 추가 설정 불필요
- 백엔드 배포 필요 (service_menu 라우터 등록)
