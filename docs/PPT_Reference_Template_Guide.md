# PPT 레퍼런스 템플릿 분석 가이드

## 목적

사내에서 "잘 만들었다"고 평가받는 PPT 샘플을 분석하여, AI PPT 생성의 콘텐츠 퀄리티·레이아웃·디자인을 실제 업무 수준에 맞게 튜닝한다.

현재 시스템 프롬프트의 좌표/패턴/규칙은 이론적으로 설정된 값이므로, 실제 레퍼런스 기반으로 보정하면 훨씬 정확한 산출물이 나온다.

---

## 필요한 레퍼런스 PPT

### 이상적인 구성 (2~3개)

| 유형 | 설명 | 분석 포인트 |
|------|------|------------|
| **보고서형** | 데이터·표 중심 (실적 보고, KPI 리뷰 등) | 테이블 구조, 여백, 데이터 밀도, 인사이트 배치 |
| **발표형** | 차트·시각요소 풍부 (전략 발표, 제안서 등) | 차트 크기/위치, 레이아웃 패턴 혼용, 색상 활용 |
| **범용형** (선택) | 혼합 구성 (교육 자료, 기획서 등) | 텍스트/표/차트 비율, 슬라이드 분배 |

### 선정 기준
- 실제 업무에서 사용된 PPT (이론적 예시 X)
- 디자인·레이아웃이 잘 되어있다고 판단되는 것
- 10장 이상 (패턴 다양성 확보)
- 민감 데이터 포함 시 → 수치를 가짜 데이터로 교체해도 됨 (레이아웃/구조만 참조)

## 분석 항목

레퍼런스 PPT에서 추출하여 시스템 프롬프트에 반영할 항목:

### 1. 레이아웃 좌표 보정
| 현재 설정 | 분석 대상 | 반영 방법 |
|-----------|----------|----------|
| body 영역 L=0.37, T=1.15, W=12.6, H=5.95 | 실제 콘텐츠 시작/끝 좌표 | 패턴별 좌표 수정 |
| shape 간 간격 0.15" | 실제 shape 간 수직 간격 | 간격 규칙 보정 |
| 하단 마진 ≤ 7.1" | 실제 하단 여백 | 마진 규칙 수정 |

### 2. 테이블 구조
- 일반적인 컬럼 수 (3~5열 vs 6열+)
- col_widths 비율 패턴 (라벨열:데이터열 비율)
- 행 수 범위 (보통 몇 행까지?)
- 헤더 스타일 (단순 1행 vs 병합 헤더)
- 본문 정렬 (중앙 vs 좌측)

### 3. 색상 활용
- 강조색 사용 빈도 (Orange/Blue 외에 다른 색?)
- 배경색 사용 패턴 (어떤 요소에 어떤 배경색?)
- 텍스트 색상 변화 (제목 vs 본문 vs 캡션)
- 차트 시리즈 색상 조합

### 4. 콘텐츠 밀도
- 슬라이드당 평균 shape 수
- 텍스트 전용 슬라이드 비율
- 불릿 포인트 평균 개수
- 여백 비율 (빈 공간 vs 콘텐츠)

### 5. 슬라이드 구성 패턴
- 슬라이드별 shape 조합 (표+텍스트? 차트만? 표+차트?)
- 간지(섹션 구분) 사용 빈도
- 목차 구조 (대/소 분류 패턴)
- 결론 슬라이드 구성

### 6. 텍스트 스타일
- 제목 폰트 크기 (현재 25pt이 적절한지)
- 본문 폰트 크기 (현재 10pt)
- 소제목/섹션 제목 스타일
- 인사이트/요약 텍스트 형태

## 분석 프로세스

```
1. 레퍼런스 PPT를 backend/data/ppt_template/reference/ 에 배치

2. template_indexer.py 확장 또는 별도 스크립트로 슬라이드별 분석:
   - 각 shape의 type, position(L/T/W/H), text 길이, font size, color
   - 테이블: 행수, 열수, col_widths 실측값
   - 차트: 타입, 크기, 위치
   - 배경색, 테두리 유무

3. 분석 결과를 패턴별로 분류:
   - "이 슬라이드는 패턴 A(테이블+인사이트)에 해당"
   - "이 슬라이드는 새로운 패턴 K에 해당"

4. 패턴별 좌표/크기 평균값 산출 → 시스템 프롬프트 반영

5. 콘텐츠 규칙 도출:
   - "테이블 뒤에 항상 1줄 인사이트가 있다" → 필수 규칙
   - "차트는 항상 전폭(12.6") 사용" → 크기 규칙
   - "불릿은 최대 4개" → 밀도 규칙

6. 시스템 프롬프트 + 좌표 업데이트
```

---

## 반영 위치

| 분석 결과 | 반영 파일 | 반영 방법 |
|-----------|----------|----------|
| 좌표/크기 보정 | ppt_worker.py (LAYOUT PATTERNS) | 패턴별 L/T/W/H 수정 |
| 테이블 규칙 | ppt_worker.py (테이블 크기 규칙) | col_widths 예시, 행 수 제한 |
| 색상 규칙 | ppt_worker.py (디자인 규칙) | 색상 사용 가이드 수정 |
| 콘텐츠 밀도 | ppt_worker.py (콘텐츠 밀도 규칙) | 불릿 수, 텍스트 비율 수정 |
| 새 패턴 | server.py + ppt_worker.py | 렌더러 추가 + 패턴 추가 |
| 스타일 상수 | template_metadata.json | font_sizes 등 수정 |
